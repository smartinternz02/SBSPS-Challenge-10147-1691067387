About the project:-

It is a campus placement prediction web application. It has the ability to

predict if a student can get a placement from his/her campus or not

based upon the student's performance. It uses a classification model

created using python. The model has been created using

Naive Bayes Algorithm and has a model accuracy of 93%.

How to use:-

 1. Fill all the details mentioned in the form.

 2. Then click on predict.
